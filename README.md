# PetAlberni

PetAlberni is a simple local pet directory homepage for Port Alberni and the Alberni Valley.

## Phase 1

This phase includes a polished, mobile-friendly static homepage with:

- PetAlberni branding
- Clean navigation
- Hero section
- Short site explanation
- Buttons for finding pet services and suggesting a business
- Category preview cards
- About section
- Contact/suggestion section
- Footer

No real business listings are included yet.

## How to view

Open `index.html` in a web browser.

## Files

- `index.html` - page content and structure
- `styles.css` - page styling and responsive layout
- `script.js` - simple mobile menu behavior

## Tech used

- HTML
- CSS
- JavaScript

No React, npm, backend, database, map API, or build tools are used.
